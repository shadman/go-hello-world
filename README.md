## Install GO
> $ sudo apt install golang-go

### Setup environment variable
> export GOPATH=$HOME/Projects/go


## Commands

### Get repository code
> $ got get github.com/shadman/go-hello-world

### Run golang program
> $ go run hello.go

### Testing
> $ go test

### Build binary (sometimes binary file is required)
> $ go build

### Run build
> $ ./helloworld

### Install binary
> $ go install
